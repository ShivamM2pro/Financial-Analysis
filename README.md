# Financial-Analysis
# 📈 End-to-End Financial Analytics & Forecasting Platform

## Overview

The **Financial Analytics & Forecasting Platform** is an end-to-end quantitative finance project designed to analyze and forecast the Indian equity market using over **21 years of historical market data (2005–Present)**.

The platform integrates **classical time-series models**, **volatility modelling**, **machine learning**, and **interactive visualization** into a unified decision-support system.

Unlike traditional dashboards that focus only on descriptive analytics, this platform combines statistical modelling with predictive analytics to provide market forecasts for the next trading session.

---

# Objectives

* Analyze long-term market behaviour of **NIFTY 50** and **SENSEX**
* Forecast next-day market direction
* Compare classical statistical models with machine learning models
* Estimate market volatility
* Deploy an interactive financial analytics application using Streamlit

# Statistical Models

## Auto ARIMA

Used for modelling long-term market trends and forecasting future price movement.

---

## GARCH

Used to estimate conditional volatility and analyse periods of market uncertainty.

---

## Random Forest

Captures nonlinear relationships within historical market features.

---

## XGBoost

Gradient boosting model used to improve predictive performance using ensemble learning.

# Statistical Diagnostics

The project incorporates statistical validation before model estimation.

Current diagnostics include:

* Normality Assessment
* Ljung–Box Test (Autocorrelation)

---

# Ensemble Learning

Instead of relying on a single model, predictions are generated using multiple forecasting approaches.

The final market outlook is obtained through an ensemble methodology combining:

* Statistical Forecasting
* Volatility Modelling
* Machine Learning
